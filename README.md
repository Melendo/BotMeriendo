# BotMeriendo - Guía de Usuario

Bienvenido a la guía de uso de **BotMeriendo**. Este bot ha sido diseñado para centralizar la música, utilidades y la diversión en tu servidor de Discord.

## Funcionalidades Principales

### 🎵 Música
*   **Reproducción**: Busca y reproduce canciones desde YouTube.
*   **Cola**: Gestión dinámica de la lista de reproducción.
*   **Desconexión Automática**: El bot se retirará si se queda solo en el canal de voz tras 1 minuto de inactividad.

### 🛠️ Utilidades
*   **Sistema de Log**: Registro detallado de actividad.
*   **Manejo de Errores**: Feedback visual para el usuario.

## Comandos

| Comando | Descripción |
| :--- | :--- |
| `!play <consulta>` | Busca y reproduce música desde YouTube. |
| `!skip` | Salta la canción actual. |
| `!stop` | Detiene la música y limpia la cola. |
| `!queue` | Muestra la lista de reproducción actual. |
| `!help` | Muestra esta ayuda y comandos adicionales. |

*(Nota: El prefijo por defecto es `!`, configurable vía variable de entorno TRGGKEY).*

## Soporte y Errores
Si encuentras un error o necesitas ayuda, revisa los logs en el canal configurado o contacta con el administrador del servidor.
